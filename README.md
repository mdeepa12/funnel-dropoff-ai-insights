## Overview
This project analyzes a customer journey funnel to identify where users drop off before completing a purchase.
It combines **SQL + Python + Dashboarding** and includes **AI-assisted insight summarization** to propose actionable improvements.

---

## Business Problem
The product team observed a decline in completed purchases and needs to understand:
- Where users are dropping off in the funnel
- Which segments (device/region) are most impacted
- What actions could improve conversion

---

## Funnel Definition
1. Visit
2. Signup
3. Product View
4. Add to Cart
5. Purchase
